# ROM-Changelog
### Install dependencies
`npm install` (in project directory)

### How to run?
* **MacOS or Linux**
`npm start`
* **Windows**
`set DEBUG=myapp:* & npm start`
* **IntelliJ IDEA (run-configuration)**
    * *Working directory:* `/your_path/rom-changelog`
    * *JavaScript file:* `./app.js`
