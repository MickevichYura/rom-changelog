# ROM-Changelog
### Install dependencies
`npm install` (in project directory)

### How to run?
* **MacOS or Linux**          
`DEBUG=myapp:* npm start`
* **Windows**      
`set DEBUG=myapp:* & npm start`
* **IntelliJ IDEA (run-configuration)**
    * *Working directory:* `/your_path/rom-changelog/bin`
    * *JavaScript file:* `www`

### Useful stuff
* **IntelliJ IDEA Plugins**
  * *NodeJS*
  * *Jade*
